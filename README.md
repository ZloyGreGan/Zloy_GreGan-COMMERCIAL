# 🎮 Юра — Java-разработчик (КОММЕРЦИЯ)
## 🔥 О себе  

Привет. Я Java-разработчик с опытом коммерческой разработки модов и игровых систем для Minecraft 1.7.10 и 1.12.2.

За 5 лет я разработал десятки модификаций: от отдельных механик и GUI до полноценных игровых систем и переработки существующего кода. Делаю упор на чистую архитектуру, производительность, расширяемость и поддержку проекта в долгосрочной перспективе.

🔹 Что я предлагаю?  

✔ Разработка модов с нуля  
✔ Доработка, рефакторинг и переписывание существующих модификаций  
✔ Реализация сложных игровых механик и кастомной логики  
✔ Разработка пользовательского интерфейса (GUI/HUD) любой сложности  
✔ Проектирование и поддержка API для интеграции между системами  
✔ Переработка игровых систем и core-логики проекта  
✔ Разработка Telegram / Discord ботов под задачи проекта  
✔ Продажа готовых технических решений и собственных наработок
