## 🔥 О себе  

Привет. Я Java-разработчик с опытом коммерческой разработки модов, плагинов и игровых систем для Minecraft 1.6.4–1.12.2.

За 5 лет разработки я реализовал десятки решений: от отдельных механик, GUI и HUD до полноценных игровых систем, API и переработки существующего кода. В работе делаю упор на чистую архитектуру, производительность, расширяемость и удобную поддержку проекта в долгосрочной перспективе.

✔ Разработка модов и плагинов с нуля  
✔ Доработка, рефакторинг и переписывание существующего кода  
✔ Реализация сложных игровых механик и кастомной логики  
✔ Разработка GUI / HUD любой сложности  
✔ Проектирование API для интеграции между системами  
✔ Переработка игровых систем и core-логики проекта  
✔ Портирование модов и плагинов между версиями 1.6.4–1.12.2  
✔ Разработка аддонов для популярных модификаций: AE2, IC2 и др.  
✔ Разработка Telegram / Discord-ботов под задачи проекта  
✔ Продажа готовых технических решений и собственных наработок
